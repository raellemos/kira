## 🛠️ COMO IMPORTAR

### Método 1: ClawHub (Recomendado)
```bash
# Instalar skill individual
clawhub install <skill-slug>

# Exemplo
clawhub install n8n-workflow-automation
clawhub install data-analyst
clawhub install google-analytics
```

### Método 2: Diretório Local
```bash
# Copiar pasta da skill para:
~/.openclaw/skills/<skill-name>/

# Estrutura:
~/.openclaw/skills/
  ├── skill-name/
  │   └── SKILL.md
```

### Método 3: Git Clone
```bash
# Para skills de repositórios externos
git clone https://github.com/best/openclaw-skills.git
cp -r openclaw-skills/<skill> ~/.openclaw/skills/
```
